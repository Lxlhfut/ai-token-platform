主流软件接入教程

选择你使用的软件，按步骤填写 Base URL 和 API Key 即可：

1
打开 Cursor 设置

点击左上角 Cursor 菜单 → Settings → Cursor Settings（或按 Ctrl+Shift+J）

2
找到 Models 页面

左侧菜单点击 Models，滚动到底部找到 「OpenAI API Key」 区域

3
填写 API Key

在 OpenAI API Key 输入框中填入你的 API Key

4
填写 Base URL（关键！）

勾选 Override OpenAI Base URL，填入 API Base URL（末尾不要加斜杠）

5
点击 Verify 验证

点击 Verify 按钮，显示绿色 ✅ 即说明配置成功！

1
打开设置页面

点击右下角「设置」图标（⚙️），进入系统设置

2
填写 API Key

在 API Key 输入框中填入你的 API Key

3
填写自定义接口地址

在 接口地址 / Custom Endpoint 填入 API Base URL（填到 /v1 这一层）

4
保存并对话

点击确认保存，返回对话界面直接开始使用即可

1
打开 Cherry Studio

进入 设置 → 模型服务（或左侧齿轮图标）

2
添加自定义提供商

点击 添加服务商，类型选择 OpenAI 兼容

3
填写配置

API 地址 填 API Base URL，API Key 填 API Key

4
添加模型

点击「检测」或手动输入模型名称（如 gpt-4o-mini），保存后即可对话

1
打开设置

左下角点击「设置」图标，进入 AI 服务商 页面

2
选择 OpenAI

在服务商列表中找到 OpenAI，点击展开设置

3
填写 API Key 和接口代理地址

API Key 填入 API Key；开启「接口代理地址」开关，填入 API Base URL

4
开始对话

返回对话页面，选择模型，即可开始使用

打开插件设置
