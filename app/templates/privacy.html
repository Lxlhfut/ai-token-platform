🔒 隐私政策

最后更新日期：2026年6月24日

1. 信息收集

1.1 当您注册平台账号时，我们会收集您的手机号码或邮箱地址，用于账号创建和身份验证。

1.2 当您使用 API 服务时，我们会记录您的 API 调用日志（包括调用的模型、Token 用量、费用等），用于计费和统计分析。

1.3 我们会记录您的充值交易记录，用于财务核对和账单查询。

2. 信息使用

2.1 您的手机号码和邮箱仅用于账号相关通知（如密码重置、重要公告等），不会用于营销推广。

2.2 API 调用日志仅用于计费和服务质量监控。

2.3 我们不会将您的个人信息出售、出租或分享给第三方，法律法规要求披露的除外。

3. 信息安全

3.1 我们采用业界通行的安全措施保护您的个人信息，包括加密存储、访问控制等。

3.2 您的登录密码经过哈希加密存储，平台工作人员无法获取您的明文密码。

3.3 请妥善保管您的 API Key，不要在公共场所或不可信的环境下使用。

4. Cookie 使用

平台使用必要的 Cookie 维持登录状态和会话管理。您可以通过浏览器设置管理 Cookie。

5. 数据存储与删除

5.1 您的个人数据存储在平台服务器上，仅用于提供和改进服务。

5.2 您可以随时联系管理员申请删除账号及关联数据。账号删除后，所有个人信息将被永久清除。

6. 未成年人保护

平台服务不面向未满 18 周岁的未成年人。如果我们发现未成年人的个人信息被收集，将及时删除。

7. 政策变更

我们可能会适时更新本隐私政策，更新后的政策将在平台公示。

8. 联系我们

如对本隐私政策有任何疑问或建议，请联系平台管理员。
