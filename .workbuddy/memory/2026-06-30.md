## 模型定价页面增强（搜索/筛选/排序/双视图）

### 改动文件
- `app/templates/dashboard.html`：pane-pricing 区域完全重写
- `app/static/app.js`：新增 10 个定价相关函数，loadUserPanel 改为调用 filterPricing()
- `app/static/style.css`：新增 `.pricing-chip` / `.pricing-chip-active` CSS 类

### 功能说明
- **搜索框**：实时输入过滤模型名/说明（`pricing-search` input）
- **排序下拉**：支持输入价↑↓、输出价↑↓、折扣↑↓、名称 A→Z（`pricing-sort` select）
- **系列筛选 chips**：动态生成（全部/GPT/Claude/DeepSeek/Gemini/Qwen/Llama/Mistral/Kimi/百川/文心/混元/MiniMax/零一/其他），点击过滤
- **视图切换**：表格视图（`pricing-table-view`）+ 卡片 grid 视图（`pricing-card-view`），右上角按钮切换
- **结果统计**：`pricing-stat` 展示「共 N 个模型」或「显示 X 个（共 N 个）」
- **卡片视图**：每个模型卡顶部彩色色条标识系列，本站价/官方价双格对比，折扣徽章，说明文字

### 核心函数
- `detectModelSeries(model)` — 识别模型系列（15种）
- `calcDiscount(p)` — 计算折扣比率
- `discountBadge(p)` — 渲染折扣徽章 HTML
- `buildPricingSeriesChips(pricing)` — 构建系列筛选 chips
- `selectPricingChip(el)` — chip 点击处理
- `switchPricingView(view)` — 切换 table/card 视图
- `setPricingSort(key)` — 列头点击排序（同列再点切换 asc/desc）
- `filterPricing()` — 主过滤+排序+渲染
- `renderPricingTable(pricing)` — 表格视图渲染
- `renderPricingCards(pricing)` — 卡片 grid 视图渲染
